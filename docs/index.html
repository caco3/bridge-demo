PebbleBridge Demo

This is the companion page for the bridge-demo Pebble watchapp, which demonstrates the native Android PebbleBridge from coredevices/mobileapp#300.

Config page

The watchapp settings open this config page:

config/index.html

What it demonstrates

The config page uses window.pebbleBridge.fetch() to call a third-party API that would normally be blocked by WebView CORS. The fetched text is sent back to the watchapp and shown on the watch.
